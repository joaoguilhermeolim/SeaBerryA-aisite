Regra de hierarquia: Máximo 1 CTA primário por ecrã (viewport). Todos os outros devem ser secundários ou ghost. Nunca dois botões verdes lado a lado.

05

Animações & Motion

↓
Fade-in ao Scroll
Secções aparecem suavemente ao entrar na viewport. Usar IntersectionObserver.
opacity: 0 → 1
translateY(24px) → 0
duration: 0.6s ease-out
delay: 0.1s por elemento
⬡
Parallax Ligeiro (Hero)
Imagem de fundo move-se a 0.4x a velocidade do scroll. Subtil, nunca exagerado.
background-attachment: fixed
ou JS: translateY(scroll * 0.4)
Apenas na secção Hero
↗
Hover em Botões
Lift subtil com sombra. Nunca escalar mais de 1.03 — parece amador.
transform: translateY(-2px)
box-shadow: 0 8px 24px ...
transition: all 0.18s ease
active: scale(0.98)
⬡
Imagens com Zoom Hover
Fotos da galeria com ligeiro zoom ao hover. Container com overflow hidden.
img { transition: transform 0.5s ease; }
img:hover { transform: scale(1.04); }
.wrap { overflow: hidden; }
⟳
Stagger de Elementos
Vários elementos entram um a um. Máximo 4 elementos no stagger.
delay: 0s, 0.1s, 0.2s, 0.3s
Nunca mais de 0.4s delay total
Usar animation-delay: Ns
✗
O que EVITAR
Qualquer animação constante/loop, bounce, rotações, parallax agressivo, pulse infinito, loading spinners decorativos.
animation-iteration-count: infinite
rotate / bounce / flip
hover scale > 1.05
Tailwind Animation Utility: Adicionar ao tailwind.config.js — fadeInUp: 'fadeInUp 0.6s ease-out forwards' e keyframe correspondente. Ou usar classe CSS custom com @keyframes fadeInUp.

07

Dicas Mobile-First

☰
Navbar Mobile
Hamburger menu em mobile. Links âncora (#hero, #sobre, etc.). Nunca abrir nova página. Fixed top com blur de fundo.
✋
Targets táteis
Todos os botões e links com min-height: 44px e min-width: 44px. Regra Apple/Google HIG para acessibilidade tátil.
↕
Hero Mobile
Usar min-height: 100svh (small viewport height) para evitar cortes do browser bar. Imagem com object-position: center.
⟶
Galeria Horizontal
Scroll snapping nativo no CSS. Esconder scrollbar com CSS. Hint visual (parte da 2ª foto visível).
A
Tamanho de Texto
Mínimo 16px em body (evita zoom automático iOS). H1 mínimo 32px em mobile. Nunca abaixo de 14px para conteúdo.
⚡
Performance
Imagens em WebP. Lazy loading nativo (loading="lazy"). Hero image prioritária (fetchpriority="high"). Max 3MB total.
☁
Imagens
Usar srcset para diferentes resoluções. Sempre definir width e height para evitar layout shift (CLS). Alt text descritivo.
↗
Links Externos
Google Maps e redes sociais abrem em nova aba (target="_blank" rel="noopener"). Botão "Como Chegar" com deep link direto.

Tailwind CSS — Configuração

tailwind.config.js

module.exports = {
  content: ['./src/**/*.{html,js}'],
  theme: {
    extend: {
      colors: {
        'purple-dark': '#3E2C56',
        'purple-mid':  '#94358F',
        'brand-green': '#497536',
        'green-hover': '#6aaa52',
        'off-white':   '#F8F7F5',
        'gray-100':    '#F0EEE9',
        'gray-500':    '#8A8680',
        'gray-700':    '#3D3A36',
      },
      fontFamily: {
        'sora':  ['Sora', 'sans-serif'],
        'inter': ['Inter', 'sans-serif'],
      },
      borderRadius: {
        'xl':   '20px',
        '2xl':  '32px',
        'pill': '999px',
      },
      animation: {
        'fade-up':     'fadeUp 0.6s ease-out forwards',
        'fade-up-d1':  'fadeUp 0.6s 0.1s ease-out forwards',
        'fade-up-d2':  'fadeUp 0.6s 0.2s ease-out forwards',
        'fade-up-d3':  'fadeUp 0.6s 0.3s ease-out forwards',
      },
      keyframes: {
        fadeUp: {
          '0%':   { opacity: '0', transform: 'translateY(24px)' },
          '100%': { opacity: '1', transform: 'translateY(0)' },
        },
      },
    },
  },
  plugins: [],
};

Scroll animation JS snippet:
const observer = new IntersectionObserver((entries) => {
  entries.forEach(e => { if(e.isIntersecting) e.target.classList.add('animate-fade-up') });
}, { threshold: 0.15 });
document.querySelectorAll('[data-animate]').forEach(el => observer.observe(el));
